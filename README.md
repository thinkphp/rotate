Rotate
======

   A simple function JavaScript that allows you to rotate images.


Syntax
======

      Rotate(HTMLImageElement image, String angle)


Usage
=====


      Rotate(document.getElementById('rotate_me'),'45deg');
      Rotate(document.getElementById('rotate_me'),Math.PI/2);
      Rotate(document.getElementById('rotate_me'),Math.PI/2);

Browser Support
===============

     Firefox 1.5+
     IE 6+
     Opera 9.5+
     Google Chrome
     Safari 2+
